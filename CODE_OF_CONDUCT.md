# Code of Conduct

This project follows the Contributor Covenant 2.1.

We are committed to providing a welcoming, harassment-free environment for
everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful in communication and review comments
- Giving constructive, actionable feedback
- Accepting and incorporating feedback professionally
- Prioritizing project health over personal ego

Examples of unacceptable behavior:

- Harassment, abuse, intimidation, or discrimination
- Personal attacks, insults, or trolling
- Publishing private information without explicit permission
- Sustained disruptive behavior in issues, PRs, or discussions

## Scope

This code of conduct applies to all project spaces, including:

- GitHub issues, pull requests, and discussions
- Project communication channels
- Any public representation of the project

## Enforcement

Maintainers are responsible for clarifying and enforcing standards.
Enforcement actions may include warning, temporary restrictions, or removal
from project spaces.

To report violations, contact maintainers privately through repository owners
or moderation channels. For security-sensitive abuse, use the private advisory
flow described in `SECURITY.md`.

All reports will be reviewed promptly and handled confidentially to the extent
possible.
