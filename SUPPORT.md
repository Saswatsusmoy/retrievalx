# Support

## Where to Ask Questions

- Usage questions: open a GitHub discussion or issue with the `question` label.
- Bug reports: use the bug report issue template.
- Feature proposals: use the feature request template.
- Security reports: follow [SECURITY.md](SECURITY.md) and report privately.

## What to Include in Support Requests

- `retrievalx` version
- Python and Rust versions
- Platform/OS details
- Minimal reproducible example
- Expected vs actual behavior

## Response Expectations

This is an open source project maintained on a best-effort basis.

- Maintainers aim to acknowledge support requests within 7 days.
- Critical bugs and security issues are prioritized.
- Well-scoped reproductions are handled fastest.

## Commercial/SLA Support

No commercial support or SLA is currently offered.
